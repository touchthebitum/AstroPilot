PROPRIÉTÉ INTELLECTUELLE ET ASPECTS LÉGAUX
Propriétaire du projet
Le projet AstroPilot est initié, dirigé et détenu par Franck Perruchoud.

Toutes les décisions fonctionnelles, techniques et stratégiques sont prises sous son autorité.

Propriété du code
Le code source développé pour AstroPilot appartient au propriétaire du projet, sauf mention contraire explicitement documentée.

Les contributions externes éventuelles devront être encadrées par une licence ou un contrat définissant clairement les droits de propriété intellectuelle.

Utilisation de ChatGPT
ChatGPT est utilisé comme assistant de développement, d'analyse et de conception.

Les idées, architectures, algorithmes, spécifications et développements intégrés dans AstroPilot sont considérés comme faisant partie du projet AstroPilot une fois sélectionnés, adaptés et implémentés par le propriétaire du projet.

Dépôt GitHub
Dépôt officiel :

https://github.com/touchthebitum/astropilot

Ce dépôt constitue la référence principale du projet.

Marque
Le nom "AstroPilot" constitue un actif stratégique du projet.

Avant toute commercialisation significative, il conviendra de vérifier :

disponibilité du nom
disponibilité du domaine internet
disponibilité des marques en Suisse
disponibilité des marques dans l'Union Européenne
Données et API tierces
AstroPilot utilise ou utilisera des données provenant de services tiers :

Open-Meteo
catalogues astronomiques
données de pollution lumineuse
données météo complémentaires
Chaque source devra être vérifiée concernant :

licence
conditions d'utilisation
limitations commerciales
Objectif long terme
Construire une plateforme AstroPilot indépendante, exploitable commercialement sur Android et iOS à l'échelle européenne puis mondiale.
